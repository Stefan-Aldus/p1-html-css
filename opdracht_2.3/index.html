Waarom heet de eerste HTML pagina index.html?

De eerste HTML pagina heet "Index.html" omdat de browser naar een file zoekt, en de eerste file die hij zoekt is index.html.

HTML

CSS

Waarom ziet een website er net iets anders uit op verschillende browsers?

De browser die je gebruikt leest de HTML en CSS file misschien anders dan een andere browser, waardoor de website er net iets verschillend uit kan zien.


Welke browser is het populairst?

De meest populaire browser is Google Chrome. Deze browser staat bekend om hoeveel RAM hij gebruikt.
Het percentage RAM dat Chrome gebruikt is afhankelijk van hoeveel tabs er open zijn en hoeveel GB RAM-Geheugen de PC heeft.
